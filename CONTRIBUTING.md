# Contributing

This project is intentionally small and focused. Contributions are welcome,
but please keep changes aligned with the original design and scope.

## About this repository

- Single-package repository
- Built with React, Next.js, Motion, and Tailwind CSS
- Distributed via npm and shadcn registry

## Development

### Fork this repo

You can fork this repo by clicking the fork button in the top right corner of this page.

### Clone on your local machine

```bash
git clone https://github.com/your-username/coverflow.git
```

### Navigate to project directory

```bash

cd coverflow
```

### Create a new branch

```bash

git checkout -b my-new-branch
```

### Install dependencies

```bash
npm install
```

### Run the project

```bash
npm run dev
```

### Guidelines

- Keep changes small and focused
- Avoid introducing new dependencies without discussion
- Maintain smooth, interruptible animations
- Follow the existing code style and structure

## Pull Requests

- Keep PRs small and focused
- Describe what changed and why
- Screenshots or recordings are encouraged for UI changes

## Code of Conduct

By participating, you agree to abide by the project's Code of Conduct.
